Les textes de narration sont tous dans les fichiers JSON du dossier :

app/content/

Plus précisément, dans chaque fichier JSON d'exemple, les textes se trouvent dans ces champs :

"text" sur les blocs "type": "diagram" → texte d'introduction de la position
"comment" sur chaque coup dans les blocs "type": "animation" → narration coup par coup
"text" sur le bloc "type": "animation" → phrase lue au lancement de l'animation
"content" (tableau de strings) sur les blocs "type": "text" → points pédagogiques à retenir
"question", "success_message", "error_message" sur les blocs "type": "quiz_interactif" → textes du quiz
