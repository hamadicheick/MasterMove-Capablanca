# Jeux de pièces (offline)

Ce dossier contient plusieurs thèmes de pièces au format SVG.

## Comment ajouter un nouveau thème
1. Crée un dossier ici: `assets/piecesets/mon_theme/`
2. Mets-y exactement 12 fichiers SVG nommés:
   - Blancs: `wp.svg wn.svg wb.svg wr.svg wq.svg wk.svg`
   - Noirs : `bp.svg bn.svg bb.svg br.svg bq.svg bk.svg`
3. Relance l’app puis sélectionne le thème dans **Paramètres → Apparence → Jeu de pièces**.

## Note
Le thème **Lichess-like** est une variante "inspirée" (100% offline) basée sur nos SVG internes.
Si tu veux le thème exact de Lichess, tu peux remplacer les 12 fichiers du dossier `lichess_like/` par ceux de ton choix en gardant les mêmes noms.
